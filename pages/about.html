About POWER Engine

POWER (Page Observer With Extraction and Route) Engine is a lightweight, template-based web engine for building dynamic interfaces without heavy frameworks.

Core Concept

  • Observe templates and page changes
  • Extract reusable UI components
  • Handle props and data cleanly
  • Navigate using a mini route system

Features

Template System

  • Multiple template extraction
  • Inline HTML injection
  • Component-style composition

Command Monitoring

<include></include>
  • Inline / internal / external CSS injection
  • Inline HTML rendering

Script Handling

  • Inline scripts
  • External script loading per route
  • Auto load and unload scripts

Props & Data

  • Dynamic prop binding
  • Structured data support

Routing & Navigation

POWER Engine uses a lightweight routing mechanism to switch pages without full reloads, keeping applications fast and responsive.

Philosophy

A web engine should be simple, observable, and transparent.

Roadmap

  • Route guards (Supported: Not yet)
  • Nested layouts
  • Scoped styles
  • Visual debugging tools (Supported: Not yet)
